# Project Rules

These are the permanent rules for the fincalclab.com project.

1. **Mobile-First Development**: Mobile-first development is mandatory.
2. **Full Responsiveness**: Every page must be fully responsive from 320px to 1920px.
3. **Priority**: Mobile responsiveness is the highest priority.
4. **Performance**: Website speed is the highest priority.
5. **SEO**: SEO is the highest priority after performance.
6. **No Placeholders**: Never use placeholder code or TODO comments.
7. **No Redesigns**: Never redesign completed sections unless explicitly requested.
8. **Astro Best Practices**: Use Astro best practices.
9. **Static Pages**: Generate static pages whenever possible.
10. **Reusable Components**: Use reusable components.
11. **Minimal JavaScript**: Keep JavaScript minimal.
12. **Production-Ready**: Always deliver production-ready code.
13. **Verification**: Test every page for desktop, tablet, and mobile before marking it complete.
